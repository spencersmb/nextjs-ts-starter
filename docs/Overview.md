## Netlify
To deploy the site. 

###Step 1
```sh
$ npm run deploy
```
This builds and then exports the site. 
<br>
<br>
###Step 2 
Add and commit changes to git, netlify will automatically rebuild the site.
```sh
$ git add .
$ git commit
$ git push
```
